# Features
* Fade effects
* Random backgrounds (there's a whopping two of them!)
* Clock
* Search bar
	* (open the search in a new tab by holding down shift)
	* (or just middle clicking the search button and then left clicking it)
* Simple premade bookmarks (for now)
